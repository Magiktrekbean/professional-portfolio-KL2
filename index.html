Cake Mix Cookies: A Fun, Interactive Recipe Website

Cake Mix Cookies is a fun, interactive website I created to showcase easy and delicious cookie recipes using cake mix as the main ingredient. The site features a clean, user-friendly design built with HTML, CSS, and JavaScript, making it simple for visitors to navigate and find their next baking project. With visually appealing imagery and an intuitive layout, the website offers step-by-step instructions and ingredient lists for each cookie recipe, ensuring a seamless user experience. The site is fully responsive and optimized for both desktop and mobile, providing a smooth browsing experience on any device. Hosted on Netlify, the site is a perfect example of my ability to craft engaging, functional, and visually appealing web experiences.

Croissant Project: A Simple Website Built with HTML, CSS, and Deployed on Netlify

I created a Git repository called "croissant-project" with three pages: `index.html`, `photo.html`, and `contact.html`. Each page is linked to one another, and the photo page includes an image of a croissant. I added some custom CSS, changing the background color for a fresh look. After deploying the site to Netlify, I made sure everything was working smoothly by redeploying it multiple times, and now the project is live on Netlify.
